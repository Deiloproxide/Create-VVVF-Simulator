# Create: VVVF-Simulator
![license](https://img.shields.io/badge/license-GPLv3-brightgreen?style=for-the-badge)<br>
机械动力：VVVF模拟器<br>
机械动力列车实时VVVF模拟
## 特性
- 实时VVVF波形模拟，无预制音频
- 通过服务器下发配置，保证远程和跨界速度的连续性
## 运行
### 预构建包

已发布到 [modrinth](https://modrinth.com/mod/create_vvvf_simulator)
### 从源码编译
1. 安装 Java 版本21及以上
2. Git 拉取仓库
```bash
git clone https://github.com/Deiloproxide/Create-VVVF-Simulator.git
```
3. 等待配置依赖
4. 通过IDE界面运行项目或通过 `./gradlew build` 构建项目
