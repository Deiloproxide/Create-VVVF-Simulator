# Create: VVVF Simulator
![许可](https://img.shields.io/badge/license-MIT-brightgreen?style=for-the-badge)

机械动力：VVVF模拟器

机械动力列车实时VVVF模拟
## 特性
- 实时VVVF波形模拟，无预制音频
## 从源码编译
1. 安装 Java 版本21及以上（Math.clamp函数需要 Java 21）
2. Git 拉取仓库
```bash
git clone https://github.com/Deiloproxide/Create-VVVF-Simulator.git
```
3. 等待配置依赖
4. 通过IDE界面运行项目或通过 `./gradlew.bat build` 构建项目
